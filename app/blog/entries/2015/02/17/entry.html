Configuring Nano
If you're anything like me all of the work you have done through SSH is getting fairly tiring when a tab seems to be a monstrous amount of space in nano. Have no fear, every user can have their own unique preferences for nano within their user folder.

My personal preferenes are:
  1. Spaces instead of tabs
  2. Tab size should be 4 spaces
  3. No case sensitivity when searching

Keep in mind that I prefer a very limited number of configurations and to view many other configurations visit the nano-editors webpage.

First step in creating the configurations is to create the configuration file .nanorc. This should be placed in your home directory (/home/{userid}/.nanorc). 
nano ~/.nanorc
Then add the following lines to the file and save. 
			set tabsize 4
			set tabstospaces
			unset casesensitive
Test it out!
