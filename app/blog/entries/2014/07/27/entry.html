Useful Tips for Blender

  • A = Select/Deselect All
  • B = Select Edges/Faces/Vertices
  • E = Extrude - Adjust single Edge/Face/Vertex
  • G = Grab - Pull on Edge/Face/Vertex affecting surroundings
  • Shift + A = Insert List
  • Shift + Middle Click = Move around in Blender
  • Space = Search for an action
  • Tab = Switch between Object and Edit Mode
